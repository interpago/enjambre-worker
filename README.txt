Enjambre Worker Node v1.0.0
=============================
Red de inferencia distribuida con IA.

¿QUE HACE?
Al ejecutarse, este worker se conecta automaticamente al enjambre.
- Si tu PC tiene GPU NVIDIA con al menos 6GB VRAM: contribuyes con
  tu GPU para procesar inferencias para toda la red (SuperNode).
- Si no tienes GPU o tiene menos VRAM: usas la API del enjambre
  como consumidor (Consumer).

USO
Ejecuta worker-node.exe. Una ventana en la bandeja del sistema
te mostrara el estado de conexion.
